# Twist game write by Python

## Installation

##### Create virtual environment:  

```bash
python -m virtualenv .venv
```


##### Use virtual environment:  
###### Linux/Unix:  
```bash
source ./.venv/bin/activate
```  
###### Windows:  
```bash
./.venv/bin/activate
```

##### Install packages with:  
  
```bash
pip install -r requirements.txt
```

## Usage 
#### Run file `main.py`
```bash
cd src
python main.py
```
